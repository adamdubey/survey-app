# surveyd-server

Backend server for `surveyd` app.

## Overview

```
.
├── LICENSE
├── README.md
├── config          # protected API keys and settings
│   └── keys.js
├── index.js
├── package-lock.json
├── package.json
├── routes          # route handlers
│   └── authRoutes.js
└── services        # helper modules and business logic
    └── passport.js
```

## Quick Start

Localhost development:

```
npm install
npm run dev
# open localhost:5000 in web browser
```
